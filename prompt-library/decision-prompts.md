# Decision-Making Prompt Library

## Purpose

These prompts help evaluate options and make informed decisions.

AI should support decisions, not make them for you.

---

# Option Comparison Prompt

Compare the following options:

[OPTION A]

[OPTION B]

[OPTION C]

Evaluate:

* Cost
* Benefits
* Risks
* Complexity
* Long-Term Value

Recommend the best option and explain why.

---

# Pros and Cons Prompt

Analyze the following decision:

[DECISION]

Provide:

* Pros
* Cons
* Assumptions
* Risks
* Unknowns

Conclude with a recommendation.

---

# Risk Assessment Prompt

Evaluate the risks associated with:

[PROJECT OR DECISION]

Identify:

* Financial risks
* Operational risks
* Reputation risks
* Strategic risks

Rank risks by severity.

Suggest mitigation strategies.

---

# Devil's Advocate Prompt

Act as a skeptical reviewer.

Critique the following idea:

[IDEA]

Identify:

* Weaknesses
* Risks
* Flawed assumptions
* Missing information

Challenge the idea aggressively but fairly.

---

# Recommendation Prompt

Act as a consultant.

Evaluate:

[OPTIONS]

Recommend the best choice.

Include:

* Supporting evidence
* Risks
* Expected outcomes
* Alternative viewpoints

Explain your reasoning step-by-step.

---

# Second Opinion Prompt

Review the following recommendation:

[RECOMMENDATION]

Attempt to find:

* Errors
* Biases
* Weak assumptions
* Missing information

Provide an independent assessment.
