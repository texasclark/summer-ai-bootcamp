# AI Tools Guide

## Purpose

This guide introduces the primary AI tools used during the Summer AI Productivity Bootcamp.

Remember:

Tools change constantly.

The goal is not mastering a specific tool.

The goal is learning how to solve problems using AI.

---

# ChatGPT

Best For:

* Brainstorming
* Research assistance
* Interactive conversations
* Project planning
* Content creation

Strengths:

* Flexible
* Easy to use
* Excellent for iterative work

Limitations:

* Can hallucinate facts
* May provide outdated information
* Requires verification

---

# Claude

Best For:

* Long documents
* Summaries
* Structured analysis
* Detailed reports

Strengths:

* Strong writing quality
* Good document handling
* Clear organization

Limitations:

* Can still hallucinate
* Requires verification

---

# Search Engines

Examples:

* Google
* Bing

Best For:

* Fact checking
* Current information
* Official sources

Use search engines whenever information must be verified.

---

# YouTube

Best For:

* Tutorials
* Demonstrations
* Industry commentary

Always evaluate creator credibility.

---

# Reddit

Best For:

* User experiences
* Community opinions
* Recommendations

Use carefully.

Opinions are not facts.

---

# GitHub

Best For:

* Project organization
* Documentation
* Portfolio building
* Collaboration

GitHub is not just for software developers.

Many professionals use GitHub to manage projects and knowledge.

---

# Choosing the Right Tool

| Goal               | Recommended Tool |
| ------------------ | ---------------- |
| Brainstorming      | ChatGPT          |
| Long Report        | Claude           |
| Fact Checking      | Search Engine    |
| Tutorials          | YouTube          |
| Community Feedback | Reddit           |
| Project Management | GitHub           |

---

# Golden Rule

Never rely on a single source.

Important information should always be verified using multiple sources.
