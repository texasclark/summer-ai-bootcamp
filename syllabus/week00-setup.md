# Week 0 – Setup and Orientation

## Objective

Create a modern AI workspace and become familiar with the tools that will be used throughout the bootcamp.

The goal is not to become software engineers.

The goal is to learn how to use AI effectively to solve problems, conduct research, automate work, and improve productivity.

---

## Learning Outcomes

By the end of this session, students will be able to:

* Sign into GitHub
* Understand what a repository is
* Install and use VS Code
* Use GitHub as a project portfolio
* Create and use ChatGPT and Claude accounts
* Complete a basic Git workflow
* Make their first commit

---

## Topics

### What is AI Productivity?

AI can help:

* Research topics
* Analyze information
* Summarize documents
* Brainstorm ideas
* Organize projects
* Automate repetitive tasks

AI cannot replace judgment, verification, or critical thinking.

---

### Understanding GitHub

GitHub is:

* A portfolio
* A project management system
* A collaboration platform
* A version history tool

GitHub is not just for programmers.

Many business professionals now use GitHub to manage projects and documentation.

---

### Understanding Repositories

A repository is simply a project folder with history.

Benefits:

* Track changes
* Organize files
* Collaborate with others
* Maintain project records

---

### Introduction to VS Code

VS Code will be used as a workspace for:

* Writing documents
* Managing projects
* Organizing prompts
* Working with AI-generated content

---

## Activity

Create a repository and add:

README.md

Include:

* Name
* Interests
* Summer goals
* One problem you would like AI to help solve

Repository with:
