# Week 2 – Personal Research Assistant

## Objective

Build an AI-assisted research system around a topic you genuinely care about.

---

## Learning Outcomes

Students will:

* Define a research objective
* Gather information from multiple sources
* Create evaluation criteria
* Produce ranked recommendations

---

## Nate Project

### Horror Game Discovery Agent

Research:

* Xbox titles
* Steam releases
* Game Pass additions
* User reviews
* Community recommendations

Possible Sources:

* Steam
* Xbox
* Reddit
* Metacritic
* YouTube

Output:

Ranked list of recommended horror games.

---

## Alaina Project

### K-Pop Performance Research Assistant

Research:

* Recent releases
* Spotify popularity
* YouTube performance
* Dance suitability
* Audience appeal

Output:

Ranked list of candidate performance songs.

---

## Research Workflow

Problem

↓

Information Gathering

↓

AI Analysis

↓

Verification

↓

Recommendation

---

## Deliverable

Recommendation Report

Include:

* Objective
* Sources
* Evaluation Criteria
* Rankings
* Justification

---

## Success Criteria

Recommendations are:

* Evidence-based
* Well organized
* Clearly explained
* Verified
