# Week 5 – Personal AI Agents

## Objective

Build a specialized AI assistant that solves a real-world problem.

---

## Learning Outcomes

Students will:

* Define agent responsibilities
* Create reusable instructions
* Improve consistency
* Generate recommendations automatically

---

## What is an AI Agent?

An AI agent is a system designed to perform a specific role repeatedly.

Examples:

* Research assistant
* Planning assistant
* Recommendation assistant
* Information monitor
