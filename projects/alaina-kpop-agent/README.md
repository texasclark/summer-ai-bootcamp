# K-Pack Show Planner

## Project Overview

K-Pack Show Planner is an AI-assisted research project designed to identify, evaluate, and recommend K-Pop songs suitable for performances and events.

The goal is to create a repeatable system that combines music trends, audience appeal, performance considerations, and research to support better performance planning.

This project focuses on learning how AI can assist with research, analysis, and decision-making.

---

## Problem Statement

Selecting songs for performances can be difficult.

Many factors must be considered:

* Audience engagement
* Popularity
* Choreography potential
* Energy levels
* Current trends

Researching all of this manually takes significant time.

The goal of K-Pack Show Planner is to streamline that process.

---

## Project Goals

By the end of the bootcamp, this project should be able to:

* Identify strong performance candidates
* Monitor current K-Pop trends
* Evaluate audience appeal
* Assess choreography suitability
* Recommend setlists
* Generate weekly planning reports
