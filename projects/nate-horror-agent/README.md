# Horror Game Scout

## Project Overview

Horror Game Scout is an AI-assisted research project designed to help identify, evaluate, and recommend horror games worth playing.

The goal is to create a repeatable process that gathers information from multiple sources, analyzes the findings, and produces ranked recommendations.

This project focuses on learning how AI can assist with research, analysis, and decision-making.

---

## Problem Statement

There are thousands of games available across multiple platforms.

Finding high-quality horror games requires:

* Research
* Review analysis
* Trend monitoring
* Community feedback

This process can be time-consuming.

The goal of Horror Game Scout is to automate much of the discovery and evaluation process.

---

## Project Goals

By the end of the bootcamp, this project should be able to:

* Discover new horror games
* Monitor Game Pass additions
* Analyze review scores
* Evaluate community sentiment
* Rank recommendations
* Generate weekly intelligence reports
