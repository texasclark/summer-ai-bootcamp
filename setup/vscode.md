# VS Code Setup Guide

## What Is VS Code?

Visual Studio Code (VS Code) is a workspace for creating and editing files.

Although developers use it for programming, we will primarily use it for:

* Writing documents
* Managing projects
* Organizing prompts
* Reviewing AI-generated content
* Working with repositories

---

## Step 1: Install VS Code

Download from:

https://code.visualstudio.com

Install using default settings.

---

## Step 2: Launch VS Code

Open the application.

You should see:

* Explorer panel
* File editor
* Activity bar

Do not worry about learning everything immediately.

---

## Step 3: Open the Bootcamp Repository

After cloning the repository:

Open Folder

Select:

summer-ai-bootcamp

The project structure should appear on the left side.

---

## Basic Features

### Explorer

View files and folders.

### Editor

Modify documents.

### Search

Search across the entire project.

### Source Control

View Git changes.

---

## Recommended Extensions

Install:

### GitHub Pull Requests and Issues

Connects VS Code to GitHub.

### Markdown All in One

Improves Markdown editing.

### Prettier

Optional formatting tool.
