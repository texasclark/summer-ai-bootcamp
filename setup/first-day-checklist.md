# First Day Checklist

Complete the following before our first bootcamp session.

## Accounts

□ Create GitHub account

□ Apply for GitHub Education

□ Create ChatGPT account

□ Create Claude account

---

## Software

□ Install VS Code

□ Install Git

---

## Repository

□ Accept repository invitation

□ Clone repository

□ Open repository in VS Code

---

## Introduction

Create a file named:

about-me.md

Include:

* Name
* Major
* Interests
* Career goals
* One problem you would like AI to help solve

Commit and push the file to GitHub.

---

## Ready for Session 1

You are ready if:

□ All accounts are working

□ Repository is accessible

□ First commit completed

□ Questions written down for discussion
