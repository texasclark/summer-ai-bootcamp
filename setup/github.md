# GitHub Setup Guide

## What Is GitHub?

GitHub is a platform for storing projects, tracking changes, and collaborating with others.

Think of GitHub as:

* A portfolio
* A project history system
* A collaboration tool
* A cloud backup for your work

Throughout this bootcamp, GitHub will be used to organize projects and track progress.

---

## Step 1: Create a GitHub Account

Visit:

https://github.com

Select:

Create Account

Choose:

* Username
* Email Address
* Password

Use an email address you check regularly.

---

## Step 2: Verify Your Email

GitHub will send a verification email.

Open the email and complete verification.

---

## Step 3: Apply for GitHub Education

Visit:

https://education.github.com

Sign in using your GitHub account.

Apply using your NCSU email address.

Benefits may include:

* Student developer tools
* Software credits
* Educational resources

Approval may take several days.

---

## Step 4: Join the Summer AI Bootcamp Repository

When invited:

1. Open the invitation email.
2. Accept the repository invitation.
3. Confirm access.

You should now see:

summer-ai-bootcamp

inside your GitHub account.

---

## Basic GitHub Terms

### Repository

A project folder.

### Commit

A saved version of your work.

### Push

Sending local changes to GitHub.

### Pull

Downloading changes from GitHub.

### Clone

Copying a repository to your computer.

---
