# Summer AI Productivity Bootcamp

Summer 2026 AI Productivity Bootcamp for Nate and Alaina

## Overview

Welcome to the Summer 2026 AI Productivity Bootcamp.

The purpose of this program is not to teach software engineering.

The purpose is to learn how to use modern AI tools to:

* Conduct research
* Analyze information
* Automate repetitive work
* Build useful personal assistants
* Make better decisions
* Improve productivity

Throughout the program students will learn how to turn real-world problems into AI-assisted workflows.

## Core Philosophy

Problem → Research → AI Assistance → Automation → Decision Making

Rather than:

Programming Language → Syntax → Homework

## Projects

### Nate – Horror Game Scout

Build an AI-powered assistant that discovers and ranks horror games using information from multiple sources.

### Alaina – K-Pack Performance Planner

Build an AI-powered assistant that identifies and evaluates K-Pop songs for performances and events.

## Weekly Topics

* Week 0 – Setup and Orientation
* Week 1 – AI Fundamentals
* Week 2 – Research Assistants
* Week 3 – Data Analysis
* Week 4 – Automation
* Week 5 – AI Agents
* Week 6 – Capstone Project

## Deliverables

Each student will complete:

* AI-assisted research reports
* Automated workflows
* Personal AI assistants
* Final capstone presentation

## Success Metric

The goal is usefulness, not code.

If a project saves time, improves decisions, or automates work, it is successful.
